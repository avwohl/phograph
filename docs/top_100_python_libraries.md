# Top 100 Python Libraries for Phograph Integration

Reference catalog of the most widely-used Python libraries, grouped by domain.
These represent the highest-value candidates for wrapping as Phograph library nodes.


## Data Science & Analysis

 1  numpy           Multidimensional arrays and fast numeric computation
 2  pandas          DataFrames for tabular data manipulation
 3  scipy           Scientific computing (optimization, linear algebra, stats)
 4  polars          Fast DataFrame library written in Rust
 5  xarray          Labeled multi-dimensional arrays (NetCDF, climate data)
 6  dask            Parallel computing on larger-than-memory datasets
 7  statsmodels     Statistical models, hypothesis tests, time series

## Machine Learning

 8  scikit-learn    Classification, regression, clustering, preprocessing
 9  xgboost         Gradient boosted trees
10  lightgbm        Fast gradient boosting (Microsoft)
